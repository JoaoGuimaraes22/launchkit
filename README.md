# next-portfolio-template

A production-ready portfolio template built with Next.js 16 App Router, React 19, TypeScript, Tailwind CSS v4, and Framer Motion.

## Features (configurable at bootstrap)

- WebGL shader hero with animated blobs and parallax
- Bilingual i18n routing (`/en`, `/pt`)
- Infinite-scroll testimonials, 2×2 services grid with modals
- Project gallery with detail pages and screenshot gallery
- Sticky profile sidebar
- Dialogflow ES chatbot
- Resend contact form
- SEO: OG/Twitter metadata, JSON-LD, sitemap, robots.txt

## Quick Start

```bash
git clone https://github.com/YOUR_GITHUB/next-portfolio-template my-portfolio
cd my-portfolio
npm install
node scripts/setup.js   # select features
# then paste BOOTSTRAP.md into a Claude Code conversation
```

See `SETUP.md` for the full setup guide and `CLAUDE.md` for architecture reference.
